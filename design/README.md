# Move the design prototypes here

This `design/` folder should hold the prototype HTML + JSX + assets from the TABOR design
project as a **read-only reference** (do not ship these; rebuild on the production stack).

Files to copy in from the design project:
- `TABOR App Prototype.html` and all `proto-*.jsx`
- `TABOR Website.html`, `site-app.jsx`, `site-store.jsx`
- `TABOR Brand Guidelines.html`, `gl-logo.jsx`, `gl-system.jsx`
- `tabor-mark.jsx`, `tokens.css`
- `TABOR Showcase.html`, `TABOR App Plan.html`, `TABOR Apparel.html`,
  `TABOR Print Catalogue.html`, `TABOR AI Prompts.html`, `TABOR Sacred Gamer Prompts.html`
- `TABOR Hub.html` (index of everything)

Open them in a browser to see the intended look and behavior. `CLAUDE.md` and `/docs`
translate them into the production build.
